================================================================================
  SetupCraft — Documentation Index
================================================================================

  Location:
    API_INTERNALS\API\         Reusable toolbox modules — public interfaces.
                               When the project is done, a consolidated API.txt
                               will be generated from these files.
    API_INTERNALS\INTERNALS\   Project-internal architecture — subsystem flow,
                               bug history, and fix guidance.

  Living document rule:
    This file is kept up to date throughout development. Every time a module
    is added, moved, or improved, the entry here is updated in the same step.
    By publish time it will be a complete, accurate index ready to seed the
    project skeleton for future Win32/C++ apps.

  Naming convention:
    _API.txt        Public component interface — what to call and what it returns.
    _INTERNALS.txt  Architectural / flow reference — how a subsystem works
                    internally, bug history, and fix guidance.

                                                and aboveAnchorY parameter for placing tooltip

                                                HideTooltip() + clear icon/tracking state in
                                                WM_LBUTTONDOWN before any ShowXxxDialog() that
                                                calls EnableWindow(parent, FALSE) — WM_MOUSELEAVE
