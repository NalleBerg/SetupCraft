================================================================================
  SetupCraft — Documentation Index
================================================================================

  Naming convention:
    _API.txt        Public component interface — what to call and what it returns.
    _INTERNALS.txt  Architectural / flow reference — how a subsystem works
                    internally, bug history, and fix guidance.

  File                              Description
  ──────────────────────────────────────────────────────────────────────────────
  tooltip_API.txt                   Custom multilingual tooltip system — creation,
                                    positioning, subclassing, and locale integration.

  spinner_dialog_API.txt            Modal progress/spinner dialog component — API,
                                    thread usage, and progress callback protocol.

  dragdrop_API.txt                  Generic Win32 TreeView/ListView drag-and-drop
                                    module — subclass-based detection, callbacks,
                                    parent hooks, and utility functions.

  files_save_load_INTERNALS.txt     Files-page save/load architecture — data flow
                                    between the TreeView, s_virtualFolderFiles,
                                    SaveTreeToDb, the DB files table, DB-rebuild
                                    load path, and fixed-bug history.

  button_INTERNALS.txt              Custom button & toolbar system — ButtonColor
                                    enum, CreateCustomButton* variants, icon
                                    property scheme, ButtonSubclassProc hover/
                                    tooltip flow, DrawCustomButton WM_DRAWITEM
                                    path, toolbar row layout, and step-by-step
                                    guide for adding a new toolbar button.

  shell32_API.txt                   Shell icon loading reference — PrivateExtractIconsW
                                    vs LoadImageW(MAKEINTRESOURCE), verified
                                    sequential icon indices for shell32.dll and
                                    imageres.dll, ImageList loading pattern,
                                    and ComponentRow dest_path cascade-scope rules.

  dialog_INTERNALS.txt              Custom modal dialog system — measure-then-create
                                    dynamic sizing pattern, layout constants,
                                    ExpandEscapes / MeasureDialogTextHeight helpers,
                                    AdjustWindowRectEx sizing recipe, WM_CREATE
                                    GetClientRect layout, and per-dialog status
                                    (which dialogs have been converted).

  edit_rtf_API.txt                  General-purpose resizable RTF editor modal —
                                    OpenRtfEditor() API, RtfEditorData struct,
                                    responsive one/two-row toolbar layout
                                    (RtfEd_LayoutToolbar, recalculated on WM_SIZE),
                                    18 controls (Bold/Italic/Underline/Strike/
                                    Subscript/Superscript/FontFace/FontSize/TextColor/
                                    Highlight/Align/Bullet/Numbered/Insert Image/
                                    Open file), Ctrl+B/I/U keyboard shortcuts,
                                    BS_AUTOCHECKBOX|BS_PUSHLIKE toggle state with
                                    EN_SELCHANGE sync, PNG/JPEG image insertion via
                                    \pict\pngblip/\jpegblip hex stream, Open button
                                    streams RTF/TXT/.md files, pLocale field for
                                    i18n (27 locale keys), hover tooltips on all 18
                                    toolbar controls (RtfEd_SetToolTip /
                                    RtfEd_ToolbarBtnProc subclass pattern), RTF stream
                                    in/out, optional char limit, WS_OVERLAPPEDWINDOW
                                    resizable, control ID range 4500–4530.

  notes_editor_API.txt              Rich-text Notes/Description editor popup —
                                    OpenNotesEditor() API, NotesEditorData struct,
                                    RTF stream in/out, 500-char limit, Bold/Italic/
                                    Underline/Subscript/Superscript/Font size/Color

  sc_shortcut_dialog_INTERNALS.txt  Configure Shortcut modal dialog — SC_EditShortcutDialog()
                                    API, ScDlgData heap struct, modal message-loop pattern,
                                    ScIconPreviewProc subclass, icon file picker filter
                                    (*.ico;*.exe;*.dll), layout constants, locale keys,
                                    and caller wiring from shortcuts.cpp.
                                    toolbar, i18n string fields, comdlg32 colour
                                    picker, DPI-aware layout.

  vfs_picker_API.txt                Reusable VFS-backed file/folder picker dialog —
                                    ShowVfsPicker() entry point, VfsPickerParams config
                                    struct (singleSelect, showFilePane, allowFolderPick,
                                    fileFilter), VfsPicker_IsExecutable() predicate,
                                    VfsPickerResult output, MainWindow::TreeSnapshot_*
                                    accessor wiring, locale keys, and usage examples for
                                    both exe-picker and folder-picker modes.

  checkbox_API.txt                  Custom theme-aware checkbox — CreateCustomCheckbox,
                                    DrawCustomCheckbox, OnCheckboxSettingChange,
                                    CreateCheckboxStateImageList,
                                    UpdateTreeViewCheckboxImages, colour palettes for
                                    Light/Dark/HC Black/HC White themes, BM_GETCHECK/
                                    BM_SETCHECK compatibility, and step-by-step guide
                                    for adding a new checkbox or TreeView state image
                                    list.

  comp_deps_INTERNALS.txt           Component dependency-summary panel — two-column
                                    ListView, DepListSubclassProc (hover tooltip via
                                    project tooltip system, double-click file popup,
                                    right-click context menu), ShowDepsFileListPopup
                                    VFS file-tree viewer, Remove/Choose custom buttons,
                                    folder-coverage filter, cascade-remove logic, and
                                    full locale key reference.

  button_rightclick_combo_INTERNALS.txt
                                    Reusable pattern: ListView + explicit action
                                    buttons + right-click context menu. Covers
                                    subclass proc wiring (SetWindowLongPtrW),
                                    TPM_RETURNCMD dispatch, Remove button enable/
                                    disable via LVN_ITEMCHANGED, context menu
                                    delegating to the button handler, tooltip
                                    integration, and step-by-step guide for
                                    replicating the pattern in a new dialog.

  shortcuts_INTERNALS.txt           Shortcuts page module architecture — static
                                    variables, SC_BuildPage / SC_Reset / SC_TearDown
                                    lifecycle, SC_OnNotify return-value convention,
                                    TreeView TVN_BEGINLABELEDIT / TVN_ENDLABELEDIT
                                    in-place rename flow, SC_OnContextMenu dispatch
                                    pattern (TPM_RETURNCMD → SendMessageW WM_COMMAND),
                                    shell32.dll icon indices used, ImageList lifetime,
                                    and full locale key table.

  deps_INTERNALS.txt                Dependencies page module architecture —
                                    ExternalDep data model (all 18 fields), enums
                                    (DepDelivery / DepArch / DepOffline /
                                    DepInstallOrder with DIO_UNSPECIFIED sentinel
                                    and DIO_CUSTOM_DIALOG stage), ListView subclass
                                    for right-click context menu, modal
                                    measure-then-create edit dialog (install order
                                    as named dropdown combo), DB table schema
                                    and CRUD API, control ID ranges, locale key table,
                                    and mainwindow.cpp wiring guide.

  scrollbar_INTERNALS.txt           Shortcuts-page vertical scrollbar — SCROLLINFO
                                    setup, why SW_SCROLLCHILDREN fails for content
                                    that starts below the viewport, correct manual
                                    child-enumeration pattern with SetWindowPos,
                                    windows to exclude (toolbar buttons, status bar,
                                    About button), SC_SetScrollOffset integration,
                                    SC_TearDown cleanup protocol, and status-bar
                                    HWND_TOP anti-overlap technique.
