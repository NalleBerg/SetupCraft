================================================================================
  SetupCraft — Documentation Index
================================================================================

  Location:
    API_INTERNALS\API\         Reusable toolbox modules — public interfaces.
                               When the project is done, a consolidated API.txt
                               will be generated from these files.
    API_INTERNALS\INTERNALS\   Project-internal architecture — subsystem flow,
                               bug history, and fix guidance.

  Living document rule:
    This file is kept up to date throughout development. Every time a module
    is added, moved, or improved, the entry here is updated in the same step.
    By publish time it will be a complete, accurate index ready to seed the
    project skeleton for future Win32/C++ apps.

  Naming convention:
    _API.txt        Public component interface — what to call and what it returns.
    _INTERNALS.txt  Architectural / flow reference — how a subsystem works
                    internally, bug history, and fix guidance.

  API_INTERNALS\API\tooltip_API.txt             Custom multilingual tooltip system — creation,
                                                positioning, subclassing, locale integration,
                                                and aboveAnchorY parameter for placing tooltip
                                                above a button (prevents blink when near screen
                                                bottom). Critical pattern: always call
                                                HideTooltip() + clear icon/tracking state in
                                                WM_LBUTTONDOWN before any ShowXxxDialog() that
                                                calls EnableWindow(parent, FALSE) — WM_MOUSELEAVE
                                                is never delivered after the parent is disabled.
                                                §16: TreeView item-hover pattern (HitTest + iImage).
                                                §17: Shared subclass proc for many dialog
                                                checkboxes/radios — attach tip string via
                                                SetPropW(L"avTip") and read in one shared proc;
                                                Win32 TOOLTIPS_CLASS unreliable for these controls.

                                                no log folder or filename is configured.

                                                block; emits one entry per ComponentRow with
                                                ComponentRow has two distinct installation
                                                behavior flags: is_required (int, DEFAULT 0)
                                                = always installed silently, hidden from wizard
                                                (component skipped in [Components] block);
                                                is_fixed (int, DEFAULT 0) = shown in wizard
                                                but greyed-out / locked, cannot be deselected

                                                DEFAULT 0) = radio-button style, selecting

                                                (never two separate Flags: entries). install_types →
                                                Types: <space-sep names>; group_name → Name:
                                                "group\comp" (backslash-separated); when any
                                                component has a group_name, an extra header
                                                row (Name: "group"; Description: "group") is

                                                migrations (14-param INSERT/UPDATE); both

                                                (IDC_COMPDLG_EXCLUSIVE=314 /
                                                IDC_FOLDER_DLG_EXCLUSIVE=329), locale key

                                                (IDC_COMPDLG_RESTART=315 /
                                                IDC_FOLDER_DLG_RESTART=330), locale key

                                                is_exclusive, is_restart, install_types + group_name on

                                                install_types/group_name from first matching
                                                file-component row.
                                                Dependencies (ComponentRow.dependencies —
                                                vector<int> of dep component IDs, persisted in
                                                component_dependencies table): shown in a ListView
                                                (IDC_COMPDLG_DEPS_LIST=309 / IDC_FOLDER_DLG_DEPS_LIST
                                                =321) with Choose (IDC_COMPDLG_CHOOSE_DEPS=310 /

                                                IDC_FOLDER_DLG_DEPS_HINT=331, shell32.dll index 23)
