================================================================================
  SetupCraft — Documentation Index
================================================================================

  Naming convention:
    _API.txt        Public component interface — what to call and what it returns.
    _INTERNALS.txt  Architectural / flow reference — how a subsystem works
                    internally, bug history, and fix guidance.

  File                              Description
  ──────────────────────────────────────────────────────────────────────────────
  tooltip_API.txt                   Custom multilingual tooltip system — creation,
                                    positioning, subclassing, and locale integration.

  spinner_dialog_API.txt            Modal progress/spinner dialog component — API,
                                    thread usage, and progress callback protocol.

  dragdrop_API.txt                  Generic Win32 TreeView/ListView drag-and-drop
                                    module — subclass-based detection, callbacks,
                                    parent hooks, and utility functions.

  files_save_load_INTERNALS.txt     Files-page save/load architecture — data flow
                                    between the TreeView, s_virtualFolderFiles,
                                    SaveTreeToDb, the DB files table, DB-rebuild
                                    load path, and fixed-bug history.

  button_INTERNALS.txt              Custom button & toolbar system — ButtonColor
                                    enum, CreateCustomButton* variants, icon
                                    property scheme, ButtonSubclassProc hover/
                                    tooltip flow, DrawCustomButton WM_DRAWITEM
                                    path, toolbar row layout, and step-by-step
                                    guide for adding a new toolbar button.

  shell32_API.txt                   Shell icon loading reference — PrivateExtractIconsW
                                    vs LoadImageW(MAKEINTRESOURCE), verified
                                    sequential icon indices for shell32.dll and
                                    imageres.dll, ImageList loading pattern,
                                    and ComponentRow dest_path cascade-scope rules.

  dialog_INTERNALS.txt              Custom modal dialog system — measure-then-create
                                    dynamic sizing pattern, layout constants,
                                    ExpandEscapes / MeasureDialogTextHeight helpers,
                                    AdjustWindowRectEx sizing recipe, WM_CREATE
                                    GetClientRect layout, and per-dialog status
                                    (which dialogs have been converted).

  notes_editor_API.txt              Rich-text Notes/Description editor popup —
                                    OpenNotesEditor() API, NotesEditorData struct,
                                    RTF stream in/out, 500-char limit, Bold/Italic/
                                    Underline/Subscript/Superscript/Font size/Color
                                    toolbar, i18n string fields, comdlg32 colour
                                    picker, DPI-aware layout.

  checkbox_API.txt                  Custom theme-aware checkbox — CreateCustomCheckbox,
                                    DrawCustomCheckbox, OnCheckboxSettingChange,
                                    CreateCheckboxStateImageList,
                                    UpdateTreeViewCheckboxImages, colour palettes for
                                    Light/Dark/HC Black/HC White themes, BM_GETCHECK/
                                    BM_SETCHECK compatibility, and step-by-step guide
                                    for adding a new checkbox or TreeView state image
                                    list.

================================================================================
