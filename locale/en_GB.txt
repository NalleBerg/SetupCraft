man_comp_p1b=If you do not need per-component selection, leave the "Use component-based installation" checkbox unticked and all files are packaged as one complete bundle — the installer skips the component-selection wizard page entirely.

man_deps_h3=⚙  Edit dialog — core fields

man_deps_h4=❓  Delivery modes
man_deps_r4a=Bundled — the installer package includes the prerequisite file. The generated script runs it silently from inside the installer.
man_deps_r4b=Auto Download — the installer fetches the file at run time from the URL you specify, verifies the SHA-256 hash, and runs it silently. Set a download timeout (seconds; 0 = wait forever) and list any acceptable non-zero exit codes in Extra exit codes.
man_deps_r4c=Redirect URL — the installer opens the URL in the default browser and pauses. The user downloads and runs the prerequisite manually; the installer waits for confirmation before continuing.
man_deps_r4d=Instructions Only — no download or file is involved. The installer displays the text from the Instructions field and asks the user to complete the step before proceeding.
man_deps_h5=🔍  Detection — registry and file checks
man_deps_p5=Before installing a prerequisite the generated installer checks whether it is already present. Two detection paths are available — either or both may be filled in:
man_deps_r5a=Registry key — an HKLM path (e.g. SOFTWARE\Microsoft\VisualStudio\14.0). If the key exists, the dep is considered installed.
man_deps_r5b=File path — an absolute path on the end user's machine (e.g. C:\Windows\System32\msvcp140.dll). If the file exists, the dep is considered installed.
man_deps_p5b=Version check: set Min version and/or Max version to restrict the acceptable version range. Choose where the version string is read from with the Version source combo: None (no version check), Registry (reads the value data of the registry key), or File (reads the FileVersion resource of the detected file).
man_deps_h6=📋  Download settings
man_deps_p6=Relevant when Delivery is Bundled or Auto Download:
man_deps_r6a=URL — the download address (Auto Download) or redirect page (Redirect URL).
man_deps_r6b=SHA-256 — hex digest of the expected installer file. The generated installer verifies this after downloading; mismatches abort.
